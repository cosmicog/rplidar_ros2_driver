# Security Policy

## Supported Versions

We only support the latest version of the code found on the main branch.

| Version | Supported          |
| ------- | ------------------ |
| `main` branch | :white_check_mark: |
| Latest Release | :white_check_mark: |
| Older Versions | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an e-mail to **frozenreboot@gmail.com**.

**Please do NOT create a public GitHub issue** for the vulnerability, as it may be exploited before a fix is available.

We will proceed as follows:
1. We will acknowledge receipt of your report as soon as possible.
2. We will investigate the issue when we have time.
3. Once resolved, we will release a patch.

Thank you for helping keep this project safe!

**Note**: This is a personal project maintained in my spare time. Response times may vary.
